# 🍳 Recipe Blog - Твоят кулинарен кът

Добре дошли в **Recipe Blog** – модерно уеб приложение за споделяне и управление на любими рецепти. Проектът е изграден с фокус върху изчистения дизайн и бързата работа.

## ✨ Функционалности
- **Преглед на рецепти:** Разгледай всички споделени кулинарни изкушения на началната страница.
- **Детайли за рецепта:** Подробна информация за всяко ястие с автор и описание.
- **Управление на съдържанието:** Възможност за добавяне, редактиране и изтриване на твоите собствени рецепти.
- **Автентикация:** Потребителска система за вход и регистрация.

## 🚀 Технологичен стек
Проектът използва най-добрите съвременни инструменти за разработка:
- **React 19** – UI библиотека за създаване на интерфейса.
- **Vite** – Мощен и изключително бърз инструмент за изграждане (Build tool).
- **React Router** – За плавна навигация между страниците.
- **JSON Server** – За съхранение и управление на данните чрез Mock API.

## 🛠️ Как да стартирате проекта локално

### 1. Клонирайте хранилището
```bash
git clone <линк-към-твоето-repo>
cd recipe-blog
```

### 2. Инсталирайте зависимостите
```bash
npm install
```

### 3. Стартирайте базата данни (JSON Server)
```bash
npx json-server --watch db.json --port 8000
```

### 4. Стартирайте приложението (Vite)
```bash
npm run dev
```
